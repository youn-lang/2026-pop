# 👥 지역별 인구구조 분석 대시보드

행정안전부 주민등록 인구통계(연령별 인구현황)를 이용하여 지역별 인구구조를 시각화하는 Streamlit 웹앱입니다.

Streamlit Cloud에서 바로 배포할 수 있도록 제작되었습니다.

---

## 주요 기능

### 1. 지역 검색 및 선택

- 시도
- 시군구
- 읍면동

모든 행정구역을 검색할 수 있습니다.

- 지역명 입력 검색
- 검색 결과 선택
- 행정구역 코드 표시

---

### 2. 연령별 인구구조 시각화

Plotly를 이용하여 다음 정보를 제공합니다.

- 전체 인구
- 남성 인구
- 여성 인구

선택적으로 표시할 수 있습니다.

또한

- 실제 인구수
- 전체 인구 대비 비율(%)

두 가지 형태로 확인할 수 있습니다.

---

### 3. 주요 인구지표

선택한 지역에 대해

- 총인구
- 남성 인구
- 여성 인구
- 65세 이상 인구
- 최다 인구 연령

을 제공합니다.

또한

- 0~14세
- 15~64세
- 65세 이상

연령집단별 인구와 비율도 제공합니다.

---

### 4. 유사 지역 분석

선택한 지역과 인구구조가 가장 비슷한 지역을 전국에서 탐색합니다.

비교 대상

- 전국 시도
- 시군구
- 읍면동

유사도 계산에는

**연령별 인구 비율(0세~100세 이상)**

을 사용합니다.

즉,

인구 규모가 아니라

연령구조 자체가 얼마나 비슷한지를 비교합니다.

제공 기능

- 유사 지역 Top 5
- 유사도 점수
- Plotly 그래프

---

### 5. 원자료 조회

연령별

- 전체
- 남성
- 여성

인구를 표 형태로 확인할 수 있습니다.
